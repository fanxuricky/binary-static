Проблемы со входом

Если у Вас возникли сложности со входом в систему, пожалуйста, убедитесь, что в браузере включены cookies. На сайте используются cookies для сохранения логина и пароля (в зашифрованном формате) на Вашем компьютере во время логин-сессии.

Для активации cookies следуйте инструкциям ниже для типа браузера, который Вы используете.

Binary (C.R.) S.A.

Chrome 51+

  • Go to "Chrome Menu (Customize and control Google Chrome)" dropdown and select Settings.
  • Нажмите кнопкуПоказать дополнительные настройки.
  • Нажмите кнопку Настройки контента вменю Приватность.
  • Select Allow local data to be set (recommended).
  • Нажмите кнопку Готово, чтобы сохранить изменения.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Откройте меню и выберитеПредпочтения.
  • Нажмите вкладку Приватность.
  • Set "Firefox will" to "Remember history".
  • Нажмите кнопку OK, чтобы сохранить настройки.
Binary (Europe) Ltd

Safari 9.x+

  • Go to the Safari menu and select Preferences.
  • Нажмите вкладку Приватность.
  • Set "Block cookies:" to Never.
  • Нажмите кнопку X в правом верхнем углу, чтобы сохранить и закрыть.
Binary (C.R.) S.A.

Opera 38+

  • Go to the Tools menu and select Preferences.
  • Нажмите на вкладку Дополнительно.
  • Select Cookies from the left column.
  • Select Accept cookies.
  • Нажмите кнопку OK, чтобы сохранить изменения.
Binary Ltd

Microsoft Internet Explorer 11+

  • Go to the Tools menu and select Internet Options.
  • Нажмите вкладку Приватность.
  • Нажмите кнопку Дополнительно.
  • Select the Override automatic cookie handling.
  • Select Accept for both First-party Cookies and Third-party Cookies.
  • Нажмите кнопку OK, чтобы сохранить изменения.
Binary Ltd

Microsoft Edge 12+

  • Go to the More actions menu and select Settings.
  • Under Advanced Settings, click on the View advanced settings button.
  • Under the Cookies section, select the "Don't block cookies" option.
  • Кликните мышкой за пределами меню, чтобы сохранить изменения и закрыть вкладку.
