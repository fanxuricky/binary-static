ログインにお困りの場合

ご口座へのログインが上手くいかない場合は、ブラウザの設定からCookieを有効に設定して下さい。当社サイトでは、ログイン時にお客さまのログインIDと暗号化されたパスワードをお客さまのコンピュータに保存する目的でCookieを利用しています。

ご利用中のブラウザのバージョンの指示に従ってクッキーを有効化して下さい。

Binary (C.R.) S.A.

Chrome 51+

  • Go to "Chrome Menu (Customize and control Google Chrome)" dropdown and select Settings.
  • Click on the Show advanced settings.
  • プライバシーメニューにあるコンテンツ設定をクリックしてください。
  • Select Allow local data to be set (recommended).
  • ”完了”ボタンをクリックし、現在の設定を保存して下さい。
Binary (IOM) Ltd

Mozilla Firefox 47+

  • メニューを開いて設定を選択してください。
  • プライバシータブをクリックしてください。
  • "Firefox"を"履歴を記憶する"に設定してください。
  • OKボタンを押してセーブし閉じて下さい。
Binary (Europe) Ltd

Safari 9.x+

  • Safariメニューへいき、設定を選択してください。
  • プライバシータブをクリックしてください。
  • ”Cookieをブロック”をNeverに設定してください。
  • 保存をして閉じるには、右上にあるXボタンをクリックしてください。
Binary (C.R.) S.A.

Opera 38+

  • ツールメニューへいき、設定を選択してください。
  • Click on the Advanced tab.
  • 左の欄にあるCookieを選択してください。
  • Cookieを有効にするを選択してください。
  • OKボタンを押してセーブし閉じて下さい。
Binary Ltd

Microsoft Internet Explorer 11+

  • ツールメニューへいき、インターネットオプションを選択してください。
  • プライバシータブをクリックしてください。
  • Click on the Advanced button.
  • Select the Override automatic cookie handling.
  • Select Accept for both First-party Cookies and Third-party Cookies.
  • OKボタンを押してセーブし閉じて下さい。
Binary Ltd

Microsoft Edge 12+

  • その他の設定メニューの設定を選択してください。
  • Under Advanced Settings, click on the View advanced settings button.
  • Cookieセクションにある"Cookieをブロックしない"オプションを選択してください。
  • 保存をして閉じるには、メニュー外をクリックしてください。
